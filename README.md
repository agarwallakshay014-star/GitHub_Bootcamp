# Freelancer Designer Landing Page

A responsive landing page built using  **HTML, CSS, and Bootstrap 5**. This project features a designer introduction section alongside an image carousel slider.

## Features

* Responsive layout using Bootstrap Grid System
* Image carousel/slider with previous and next controls
* Clean and modern design
* Hover effects on buttons
* Images automatically fit within the slider using `object-fit: cover`
* Vertically and horizontally centered content

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5.3

## How to Run

1. Download or clone the repository.
2. Place all image files in the project folder.
3. Open `index.html` in your web browser.
